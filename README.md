# bernardojeronimo.github.io

Personal portfolio website for the **Desenvolvimento de Interfaces Web** (Web Interface Development) course at [Universidade Lusófona](https://www.ulusofona.pt/), 2024.

🔗 **Live site:** [https://bernardojeronimo.github.io](https://bernardojeronimo.github.io)

## About

This repository hosts a collection of lab exercises and project defenses completed throughout the Web Interface Development course. Each lab progressively builds web development skills, starting from basic HTML through advanced CSS and JavaScript.

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Media Queries, Animations)
- JavaScript (ES6+)

## Project Structure

```
├── index.html          # Main landing page with links to all labs
├── styles.css          # Global stylesheet
├── lab1/               # Lab 1 – My first website online
├── lab2/               # Lab 2 – HTML5 – Cities
├── lab3/               # Lab 3 – HTML – City with Forms
├── lab4/               # Lab 4 – CSS – Styled city with selectors
├── lab5/               # Lab 5 – CSS Part 2 – City
├── lab6/               # Lab 6 – CSS (Grid / Media Queries / Animations)
├── lab7/               # Lab 7 – JavaScript Part 1
├── lab8/               # Lab 8 – JavaScript Part 2
├── lab9/               # Lab 9 – Store Part 1
├── lab10/              # Lab 10 – Store Part 2
├── DEFESA P1/          # Project Defense 1
└── defesa2/            # Project Defense 2
```

## Labs Overview

| Lab | Topic | Key Concepts |
|-----|-------|--------------|
| 1 | First Website | Basic HTML structure, images |
| 2 | HTML5 – Cities | Semantic HTML, multi-page navigation |
| 3 | HTML – Forms | Forms, input elements, validation |
| 4 | CSS – Selectors | CSS selectors, styling fundamentals |
| 5 | CSS Part 2 | Advanced layouts, positioning |
| 6 | CSS Advanced | CSS Grid, media queries, animations |
| 7 | JavaScript Part 1 | DOM manipulation, events |
| 8 | JavaScript Part 2 | Advanced JavaScript, interactivity |
| 9 | Store Part 1 | Dynamic product rendering |
| 10 | Store Part 2 | Shopping cart, local storage |

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/bernardojeronimo/bernardojeronimo.github.io.git
   ```
2. Open `index.html` in your browser, or navigate into any lab folder and open its `index.html`.

No build tools or dependencies are required — the project uses only vanilla HTML, CSS, and JavaScript.

## Deployment

The site is automatically deployed via [GitHub Pages](https://pages.github.com/) from the default branch.
